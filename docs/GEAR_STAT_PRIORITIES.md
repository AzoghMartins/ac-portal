# Gear Stat Priorities
For AzerothCore 3.3.5a
Client Build 12340

This document defines stat priorities for curated Uncommon (Green) gear sets.

Design Philosophy:
- Represents "Just Dinged" characters.
- No min-max optimization.
- No hit-capping.
- No crit stacking.
- No burst scaling.
- Survivable and functional for entry-level content.

------------------------------------------------------------
LEVEL 60 (VANILLA ITEMS ONLY, ILVL ≤ 65 (as close to 65 as possible), ReqLVL ≤ 60 (as close to 60 as possible))
------------------------------------------------------------

All stats follow Vanilla itemization structure (Defense skill, not rating).
